什么是Markdown ?

在准备学习一门新玩意儿的时，至少要搞清楚它是什么，自己真的需要它吗 ?

Markdown 是一种轻量级标记语言，创始人为约翰·格鲁伯（John Gruber）。
它允许人们“使用易读易写的纯文本格式编写文档，然后转换成有效的XHTML(或者HTML)文档”。
[4]这种语言吸收了很多在电子邮件中已有的纯文本标记的特性。——维基百科

通俗的说，Markdown是用于协作，那么它相对其他写作产品，比如Word，有什么优势呢，以及这些优势是否是你所需要的呢 ?

  1. 在整个写作中，你不需要用鼠标去点那些“加粗”，“字号”，“加链接”等等功能，在Markdown中手打一到两个符号就代表加粗，斜体等等功能。
  2. 它会帮你做一些基础排版，让你更加的专注于写作。
  3. Markdown编辑器启动速度更快，而且界面更加的清爽直观。
  4. Markdown文件可以上传Github上（意味着可以共同协作），而Word文件是二进制文件，无法上传。

Markdown在网上似乎炒得很热，觉得是一种趋势，我个人觉得Markdown并没有革命性意义的。如果作者想写谋篇文章，完全可以去印象笔记，网易云协作，石墨文档等来完成，
而且它们没有任何学习难度，零成本。Markdown至少还是存在一点成本。我认为Markdown不论在现在，还是将来，它是一个小众软件，做不到像Word那样，它更多会用在我这样写博客的人。
